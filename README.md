# RDDA

## Dependency

### Install SOEM
`git clone https://github.com/OpenEtherCATsociety/SOEM`

`cd SOEM`

`mkdir build`

`cd build`

`cmake .. -DCMAKE_INSTALL_PREFIX=/usr/local/`

`make`

`sudo make install`

## Install RDDA
`git clone https://github.com/RoboticsCollaborative/RDDA.git`

`mkdir build`

`cd build`

`cmake ..`

`make`
